# VoiceDesk

Adobe Premiere Pro のパネルから **AquesTalk / VOICEVOX(互換エンジン含む) / A.I.VOICE2** を
まとめて操作できるエクステンションです。掛け合い台本をそのまま音声化して、
タイムラインへの配置・字幕生成まで行えます。

![panel] (パネルはPremiereのウィンドウ > エクステンション > VoiceDesk に表示されます)

## 対応ソフト

| ソフト | 方式 | 備考 |
|---|---|---|
| AquesTalkPlayer | コマンドライン連携 | プリセット(声種)を自動読込 |
| VOICEVOX | ローカルAPI | 全話者・全スタイル対応、エンジン自動起動 |
| VOICEVOX互換エンジン | ローカルAPI | AivisSpeech、SHAREVOX等。複数登録可 |
| A.I.VOICE2 | UI自動操作(同梱ブリッジ) | 公式API公開までの暫定方式 |

## インストール

1. zipを展開し `install.bat` をダブルクリック
   (%APPDATA%\Adobe\CEP\extensions へコピーし、未署名エクステンションを許可します)
2. Premiere Pro を再起動
3. メニュー: ウィンドウ > エクステンション > VoiceDesk

## 初期設定(使うものだけでOK)

「設定」欄を開いて設定します。すべて自動保存されます。

- **共通**: WAV出力フォルダ / txtの文字コード(既定: Shift-JIS)
- **AquesTalk**: AquesTalkPlayer.exe のパス(参照ボタンで選択可)
- **VOICEVOX/互換エンジン**: 既定でVOICEVOX(ポート50021)が1件登録済み。
  「＋エンジンを追加」で AivisSpeech(10101) 等を追加。
  実行ファイルを設定しておくと未起動時に自動起動します
- **A.I.VOICE2**: aivoice.exe のパスと書き出し先フォルダ

## 使い方

### 掛け合いエディタ
1行 = 「声 / トラック / セリフ」。＋で行を追加、×で削除。

- **声**: [AQ]=AquesTalk、[エンジン名]=VOICEVOX系、[A.I.VOICE2] が1つのリストに並びます
- **A**: その声の配置先オーディオトラック(声ごとに記憶されます)
- **▶**: その行だけ試聴(VOICEVOX系はパネル内再生で画面を奪いません)
- **💾**: その行だけを保存(WAV+txt生成。「保存時に音声をシーケンスへ配置」がONならその行も
  配置)。保存済みの行はグレー表示になり、声やセリフを編集すると解除されます
- 複数行テキストを貼り付けると自動で行に分割されます

### 全行を保存
上から順にWAV(+字幕用txt)を生成します。出力フォルダの直下ではなく、
声(キャラ)ごとのサブフォルダが自動的に作られ、その中に保存されます。

- フォルダ名は声の種類で決まります: AquesTalkは`AQ_<プリセット名>`、
  VOICEVOX系は`<エンジン名>_<キャラ名>`(スタイル(ノーマル/あまあま等)が
  違っても同じキャラなら同じフォルダにまとまります)、A.I.VOICE2は`AIVOICE2`
- ファイル名の先頭には、そのフォルダ内だけで独立した3桁連番(`001_`, `002_`…)が
  付きます。連番はフォルダ内の既存ファイルの続きから採番されるため、
  既存ファイルを上書きすることはありません
- 「ファイル名の先頭にキャラ名を付ける」をONにすると、連番の後にキャラ名が続きます
  (順序: 連番 → キャラ名(ON時のみ) → セリフ)。OFFの場合は連番の後にセリフが続きます
- 「保存時に音声をシーケンスへ配置」ON: 再生ヘッド位置から順に、
  声ごとに設定したトラックへ自動配置します。インポート先のVoiceDeskビンも
  声ごとのサブビンに分かれます
- A.I.VOICE2の行は書き出しフォルダを監視して完了を検知します
- 保存に成功した行はグレー表示になります。「使用済みのセリフを削除」ボタンで、保存済みの行の
  セリフだけをまとめて空にできます(行・声・トラック設定はそのまま残ります)

### ツール: 選択クリップから字幕を一括生成
タイムラインで音声クリップを選択(未選択ならターゲット中のオーディオトラック全体)して実行。
各クリップと同名のtxtを読み、実際の位置・長さに合った字幕を
**1本のキャプショントラック**として挿入します。

## トラブルシューティング

- **声リストに出ない** → エンジンを起動して、掛け合いエディタ右上の↻を押す
- **パネルが表示されない** → install.bat を再実行しPremiereを再起動。
  それでも出ない場合はレジストリ `HKCU\Software\Adobe\CSXS.11` の
  `PlayerDebugMode=1`(文字列)を確認
- **A.I.VOICE2でエラー** → A.I.VOICE2を起動した状態で実行。
  aivoice.exe のパスが未設定・誤っている場合もエラーになるため、設定欄のパスを確認してください。
  UIが更新で変わった場合は bridge/aivoice2_bridge.ps1 の調整が必要です
